VideoDisplay class

Description

VideoDisplay Class handles displaying video streams and handles switching to new streams.

Method summary for the VideoDisplay class

Method Description
VideoDisplay.setMedia()

Method creates the video object and connects the NetStream refference to display.
VideoDisplay.updateVideo()

Method updates the video display with a new netstream

Property summary for the VideoDisplay class

Property Description
VideoDisplay.connectionManager

Method sets the connectionManager object to handle displaying net streams.
VideoDisplay.setHeight

Method sets the height of the video object.
VideoDisplay.setWidth

Method sets the width of the video object.
VideoDisplay.smoothing

Sets the video smoothing property.
VideoDisplay.deblocking

Sets the video deblocking value.
