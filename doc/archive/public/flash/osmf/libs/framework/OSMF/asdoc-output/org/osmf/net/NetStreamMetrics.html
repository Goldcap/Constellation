Classpublic class NetStreamMetrics
InheritanceNetStreamMetrics NetStreamMetricsBase flash.events.EventDispatcher

The NetStreamMetrics class makes use of the metrics offered by NetStream.info, but more importantly it calculates running averages, which we feel are more robust metrics on which to make switching decisions.
